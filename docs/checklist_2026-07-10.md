# 2026-07-10 변경사항 점검 체크리스트

테스트 환경: http://localhost:5173
테스트 계정: 조직 관리자 또는 일반 멤버 계정

---

## 1. 시간별/일자별 보기 모드 토글

배경: 월간·주간 뷰에서 시간축 표 대신 날짜별 배정 요약 목록으로 전환할 수 있는 토글을 추가했다.

### 월간 뷰

- [ ] 월간 탭에서 헤더 우측에 "⏳ 시간별 / 📅 일자별" 토글이 표시되는가
- [ ] "📅 일자별" 클릭 시 날짜별 요약 목록으로 전환되는가
- [ ] 날짜별 요약에서 각 날짜 셀에 배정 항목이 시간순으로 표시되는가
- [ ] 배정이 없는 날짜는 "-"로 표시되는가
- [ ] 배정이 3건 초과인 날짜에서 "+N건 더" 버튼이 표시되고 클릭 시 펼쳐지는가
- [ ] 펼쳐진 상태에서 "접기" 클릭 시 다시 접히는가
- [ ] 배정 항목 클릭 시 기존 수정 모달(SlotEditModal)이 정상적으로 열리는가
- [ ] "⏳ 시간별" 클릭 시 기존 ScheduleGrid 표로 돌아오는가
- [ ] 공휴일 날짜의 배경 색상(빨간 음영)이 일자별 뷰에서도 적용되는가

### 주간 뷰

- [ ] 주간 탭 전환 후 헤더에 "⏳ 시간별 / 📅 일자별" 토글이 표시되는가
- [ ] "📅 일자별" 클릭 시 7컬럼 날짜별 요약으로 전환되는가
- [ ] 오늘 날짜 컬럼에 테두리 강조(ring)가 표시되는가
- [ ] 오늘 날짜 컬럼으로 자동 스크롤되는가
- [ ] 배정 항목 클릭 시 기존 수정 모달이 정상적으로 열리는가
- [ ] "⏳ 시간별" 클릭 시 기존 WeekGrid 표로 돌아오는가

### 일간 뷰

- [ ] 일간 탭으로 전환 시 헤더에 토글이 표시되지 않는가

### 엑셀 모드

- [ ] 엑셀 모드 활성화 시 "⏳ 시간별 / 📅 일자별" 토글이 사라지는가
- [ ] 엑셀 모드 비활성화 시 토글이 다시 나타나는가

### 역할 분리(Split) 모드

- [ ] 역할 분리 조직에서 일자별 뷰 진입 시 배정 항목 앞에 `[역할명]`이 표시되는가
- [ ] 역할 토글(역할 표시 버튼)로 특정 역할 숨김 시 해당 역할 배정이 일자별 뷰에서도 사라지는가

## 2. 회귀 테스트 (기존 기능 영향 확인)

- [ ] 월간 뷰 "⏳ 시간별" 상태에서 기존 ScheduleGrid 동작(셀 클릭, 배정 추가)이 정상인가
- [ ] 주간 뷰 "⏳ 시간별" 상태에서 기존 WeekGrid 동작이 정상인가
- [ ] SharePage(비회원 공유 링크)에 토글이 노출되지 않는가
- [ ] EmbedPage(외부 임베드)에 토글이 노출되지 않는가
- [ ] 페이지 새로고침 후 "⏳ 시간별"(기본값)로 초기화되는가

---

## 테스트 우선순위

1. **1번 — 월간 뷰** 토글 전환 및 배정 항목 클릭 (핵심 신규 기능)
2. **1번 — 주간 뷰** 토글 전환 및 오늘 날짜 강조
3. **2번 — 회귀** SharePage/EmbedPage에 토글 미노출 확인
