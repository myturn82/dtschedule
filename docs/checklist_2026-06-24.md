# 변경사항 점검 체크리스트 — 2026-06-24

## 고객계정 멀티선택 삭제 (AccountRail)

- [ ] AccountRail 헤더에 "선택" 버튼이 표시됨
- [ ] "선택" 클릭 시 각 고객 행에 체크박스 표시
- [ ] 고객 클릭 시 체크박스 토글 (페이지 이동 안 됨)
- [ ] 1개 이상 선택 시 하단에 "N개 선택됨 | 삭제" 표시
- [ ] "취소" 클릭 시 선택 모드 해제, 체크박스 사라짐
- [ ] "삭제" 클릭 시 모달 표시 (고객 목록 + 조직 수 표시)
- [ ] "비활성화" 클릭 → 고객 상태 비활성으로 변경 (목록에서 회색 처리)
- [ ] "영구 삭제" 클릭 → 고객이 목록에서 완전 제거
- [ ] 시스템 고객 계정은 삭제 목록에서 선택 불가 (해당 시)

## 사용자 관리 탭 (UserManagementPanel)

- [ ] "사용자 관리" 탭 클릭 시 화면 전환
- [ ] 사용자 목록에 이름, 이메일, 가입 조직 수, 가입일 표시
- [ ] 조직 0개 사용자에 "미가입" (amber) 뱃지 표시
- [ ] 슈퍼관리자 계정에 "슈퍼관리자" 뱃지 + 체크박스 비활성화
- [ ] 이름/이메일 검색 동작
- [ ] "미가입만" 필터 클릭 시 조직 0개 사용자만 표시
- [ ] 체크박스 멀티선택 및 "전체 선택" 체크박스 동작
- [ ] "N명 선택됨 | 선택 삭제" 액션바 표시
- [ ] 삭제 모달 — 선택 사용자 목록 + 조직 수 표시
- [ ] "영구 삭제" 클릭 → 사용자 목록에서 제거
- [ ] 탭 재진입 시 재로딩 없이 캐시 유지
