## 1. 이메일 회원가입 위자드 — 전화번호 입력 단계 추가

배경: 가입 시 전화번호를 수집해 `profiles.phone`에 저장할 수 있도록 위자드에 phone 단계를 추가했다.
confirm 단계 다음에 phone 단계가 삽입되며, 입력은 선택 사항이다.

### 위자드 흐름

- [ ] 이메일 가입 위자드 진행 시 confirm(비밀번호 확인) 이후 **phone(전화번호) 단계**가 표시되는가
- [ ] phone 단계에서 전화번호 미입력 후 "계속하기" → choice 단계로 정상 진행되는가
- [ ] phone 단계에서 유효한 번호(010-1234-5678) 입력 후 "계속하기" → choice 단계로 정상 진행되는가
- [ ] 잘못된 형식(예: 0101234) 입력 후 "계속하기" 클릭 → 오류 메시지가 표시되는가
- [ ] phone 단계에서 "뒤로" → confirm 단계로 돌아가는가
- [ ] 위자드 닫기(X) → phone 입력값이 초기화되는가

### 가입 완료 후 DB 반영

- [ ] **[DB 확인]** 전화번호를 입력하고 가입 완료 → Supabase `profiles` 테이블의 `phone` 컬럼에 입력값이 저장되는가
- [ ] **[DB 확인]** 전화번호를 입력하지 않고 가입 완료 → `profiles.phone`이 `null`인가
- [ ] "내 서비스 시작하기" 경로(org-name 단계 포함)에서도 `phone`이 올바르게 저장되는가
- [ ] "기존 조직에 가입하기" 경로(org-select 단계)에서도 `phone`이 올바르게 저장되는가

## 2. 위자드 진행 표시 (dots) 확인
