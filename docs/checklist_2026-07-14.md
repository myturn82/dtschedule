## 1. 일자별 요약 뷰(월간·주간) 스케줄 등록 개선

배경: 월간/주간 "일자별 보기 모드"(`MonthScheduleByDay`, `WeekScheduleByDay`)에서는 기존 배정을 클릭해 수정하는 것만 가능하고, 새 스케줄을 원하는 시간대로 등록할 방법이 전혀 없었음(배정이 없는 날은 "-"만 표시). 각 날짜 셀에 "+ 등록" 드롭다운을 추가해 해당 날짜의 운영 중인 시간대 중 원하는 시간을 선택해 바로 등록 팝업이 뜨도록 개선.

- [ ] 월간 뷰 → 일자별 보기 모드에서, 배정이 없는 날짜 셀에 "+ 등록" 드롭다운이 보임 (휴관/휴게 등으로 운영 시간대가 아예 없는 날은 기존처럼 "-" 표시)
- [ ] "+ 등록" 드롭다운을 열면 해당 날짜에 운영 중인 시간대 목록이 나옴 (휴관/휴게 시간대는 목록에서 제외됨)
- [ ] 원하는 시간대를 선택하면 그 시간대로 스케줄 등록 팝업이 뜸
- [ ] 이미 배정이 있는 날짜 셀에서도 "+ 등록"으로 다른(비어있는) 시간대에 추가 등록 가능
- [ ] 기존 배정 항목을 클릭하면 여전히 해당 배정 수정 팝업이 뜸 (회귀 없음)
- [ ] 주간 뷰 → 일자별 보기 모드에서도 동일하게 동작
- [ ] 비회원(방문자 예약) 모드처럼 `canAdd`가 false인 경우 "+ 등록"이 보이지 않음
- [ ] 다크모드에서 "+ 등록" 드롭다운 팝업이 정상적으로 보임 (이전 다크모드 select 수정 회귀 없음)
