## 1. 커스텀 입력필드 타입 확장 (AdminPage → 입력필드 탭)

배경: 기존 text/select 2가지 타입에서 number, radio, checkbox, checkbox_group, phone, account_number 까지 8가지로 확장

### 1-1. 필드 추가

- [ ] 관리자 콘솔 → 입력필드 탭 진입
- [ ] **텍스트** 필드 추가 — 플레이스홀더 설정 후 저장, 카드 미리보기에 `<input>` 표시 확인
- [ ] **숫자** 필드 추가 — 최솟값/최댓값 설정 후 저장, 카드 미리보기에 `<input type="number">` 확인
- [ ] **드롭다운** 필드 추가 — 옵션 2개 이상 추가, 대시보드 포함 체크, 저장 확인
- [ ] **라디오** 필드 추가 — 옵션 추가, 카드 미리보기에 라디오 버튼 표시 확인
- [ ] **체크박스** 필드 추가 — 단일 체크박스, 대시보드 포함 설정 확인
- [ ] **다중선택** 필드 추가 — 옵션 추가, 카드 미리보기에 체크박스 그룹 표시 확인
- [ ] **전화번호** 필드 추가 — 카드 미리보기에 `010-0000-0000` placeholder 확인
- [ ] **계좌번호** 필드 추가 — 카드 미리보기에 계좌번호 placeholder 확인

### 1-2. 카드 UI

- [ ] 필드 카드 전체 클릭 → 인라인 편집 진입 (기존 "수정" 버튼 없음)
- [ ] 편집 진입 시 카드 테두리가 brand 색으로 변경 확인
- [ ] 타입 배지 색상: text=slate, number=blue, select=emerald, radio=orange, checkbox=purple, checkbox_group=pink, phone=teal, account_number=violet
- [ ] ↑↓ 순서 이동 버튼 동작 확인
- [ ] 삭제 버튼 → 확인 팝업 후 삭제 확인

### 1-3. 편집 폼 타입별 조건부 섹션

- [ ] **숫자** 타입 선택 시 최솟값/최댓값 입력란 표시
- [ ] **드롭다운/라디오/다중선택** 타입 선택 시 옵션 관리 + 대시보드 통계 포함 체크박스 표시
- [ ] **체크박스** 타입 선택 시 대시보드 통계 포함만 표시 (옵션 없음, 필수 체크박스 없음)
- [ ] **전화번호/계좌번호** 타입 선택 시 플레이스홀더 입력란 표시

- [ ] **[DB 확인]** `tenants.settings.custom_fields` 배열에 type, min, max, options, show_in_dashboard 값이 올바르게 저장됨

## 4. 대시보드 — 사용자 정의 통계 위젯

배경: 관리자가 원하는 필드 조합으로 통계 위젯을 직접 구성하는 기능 추가

### 4-1. 위젯 추가

- [ ] 관리자 계정으로 대시보드 접근 → "+ 통계 위젯 추가" 점선 버튼 표시 확인
- [ ] 일반 회원 계정 → 위젯 추가 버튼 비표시 확인
- [ ] 버튼 클릭 → 위젯 설정 모달 오픈

### 4-2. 모달 설정 — 건수(COUNT) 위젯

- [ ] 위젯 제목 입력
- [ ] 집계 기준 필드 선택 (드롭다운 또는 라디오 필드)
- [ ] "건수 COUNT" 버튼 선택
- [ ] 막대 차트 선택 후 저장
- [ ] 대시보드에 위젯 카드 표시, 막대 차트 렌더링 확인
- [ ] **[DB 확인]** `tenants.settings.dashboard_widgets` 배열에 위젯 설정 저장 확인

### 4-3. 모달 설정 — 합계(SUM) 위젯

- [ ] 집계 기준: 드롭다운 필드 선택
- [ ] 집계 방식: 숫자 필드 "합계" 버튼 선택
- [ ] 파이 차트 선택 후 저장
- [ ] 파이 차트와 범례(레이블 + 건수 + 비율%) 표시 확인

### 4-4. 모달 설정 — 평균(AVG) 위젯

- [ ] 집계 기준: 라디오 필드 선택
- [ ] 집계 방식: 숫자 필드 "평균" 버튼 선택
- [ ] 표(table) 유형 선택 후 저장
- [ ] 항목별 값 + 비율% 표시 확인

### 4-5. 차트 유형 확인

- [ ] 막대 차트: 최댓값 항목이 100% 너비, 나머지 비례 렌더링
- [ ] 파이 차트: SVG 조각이 정상 렌더링, 합계가 1건일 때 360도 원형 확인
- [ ] 표: 항목별 행 표시, 맨 아래 합계 카드

### 4-6. 위젯 수정 / 삭제

- [ ] 관리자 계정에서 위젯 카드의 "수정" 버튼 → 기존 설정 로드된 모달 오픈
- [ ] 수정 후 저장 → 위젯 내용 업데이트 확인
- [ ] "삭제" 버튼 → 확인 팝업 후 위젯 제거 확인
- [ ] **[DB 확인]** 삭제 후 `dashboard_widgets` 배열에서 해당 위젯 제거됨

### 4-7. 데이터 없을 때

- [ ] 해당 월 집계 데이터 없는 위젯 → "이 달 집계 데이터가 없습니다" 문구 표시

## 7. 모바일 UI 개선

### 7-1. 대시보드 모바일 반응형

- [ ] 모바일에서 막대 차트 → 긴 값("12건(240,000원)")이 화면 밖으로 벗어나지 않음
- [ ] 모바일에서 파이 차트 범례 → 값·퍼센트가 세로로 쌓여 오버플로우 없음
- [ ] 모바일에서 표 차트 → 값·퍼센트가 세로로 쌓여 오버플로우 없음
- [ ] 위젯 카드 헤더 집계 라벨(aggLabel)이 길어도 truncate 처리 확인

### 7-2. 스케줄 역할명 모바일 overflow 수정

- [ ] 월간 뷰(split 모드) — 역할명이 셀을 벗어나지 않고 말줄임 처리 확인
- [ ] 주간 뷰 — 역할 서브헤더 역할명이 말줄임 처리 확인
- [ ] 일간 뷰 — 역할명이 말줄임 처리 확인
- [ ] (공통) 역할명 길 경우 마우스 오버 시 title 속성으로 전체 이름 확인 가능

### 7-3. 스케줄 화면 모바일 레이아웃 최적화

- [ ] 모바일에서 달력 영역 테두리/그림자 없음, 화면 폭 100% 활용 확인
- [ ] 모바일에서 달력 배경이 헤더(`--color-surface`)와 동일한 색상 확인 (변경 전: `--color-bg`)
- [ ] 모바일에서 헤더 내부 패딩 축소 확인 (데스크탑은 기존 유지)
- [ ] 모바일에서 달력 셀 패딩 축소 확인 (헤더 셀, 시간 라벨, WeekGrid 셀)
- [ ] 데스크탑에서 기존 카드 스타일(배경·테두리·라운드·그림자) 그대로 유지 확인
