## 14. 슈퍼관리자 고객 헤더(요금제/가입일/전화번호/오너) 정보 카드로 디자인 변경

배경: HubMain.tsx의 고객 헤더 정보 행(요금제 선택, 전화번호, 오너, 가입일)이
인라인 배지/2열 그리드 형태일 때 좁은 영역에서 값이 말줄임(...) 처리되어
전화번호·이메일 일부가 가려지는 문제가 있었음. "라벨 + 값"을 각각 한 줄 전체 너비로
세로로 쌓는 형태로 디자인을 변경하고 말줄임을 제거, 값이 길면 줄바꿈되어 전체 텍스트가
항상 보이도록 함 (이메일은 `break-all`로 필요 시 단어 중간에서도 줄바꿈).

- [ ] 슈퍼관리자 화면 → 고객 선택 → 헤더에 "요금제 / 가입일 / 전화번호 / 오너"가
      각각 라벨 + 값 형태로 한 줄씩 세로로 표시됨
- [ ] 전화번호 값(`010-XXXX-XXXX`)이 말줄임 없이 전체가 보임
- [ ] 오너 이메일이 말줄임 없이 전체가 보임 (너비를 넘으면 줄바꿈되어 표시, 잘리지 않음)
- [ ] 요금제(`Business` 등)와 가입일(`YYYY-MM-DD`)이 잘리지 않고 전체가 표시됨
- [ ] 전화번호 클릭 → 편집 모드 전환 시 입력창+저장/취소 버튼이 정상적으로 표시·동작함
      (저장/취소/Enter/Esc 모두 정상)
- [ ] 오너 클릭 → 편집 모드 전환 시 동일하게 입력창+저장/취소 버튼이 정상 동작함
- [ ] 요금제 선택(Select) 변경 시 정상 반영됨
- [ ] 모바일/데스크톱 화면 폭 전환 시 레이아웃이 깨지지 않음
