## 1. 커스텀 필드 textarea 자동 높이 확장

배경: 커스텀 필드 입력란이 `input`으로 되어 있어 긴 텍스트 입력 시 스크롤되거나 잘렸고, 읽기모드에서도 줄바꿈이 표시되지 않았음.

### AdminPage — 커스텀 필드 탭

- [ ] 커스텀 필드 편집 모드에서 필드명 입력란이 한 줄로 시작하는가
- [ ] 필드명에 긴 텍스트 입력 시 입력란 높이가 자동으로 늘어나는가
- [ ] 엔터(줄바꿈) 입력 시 높이가 늘어나는가
- [ ] "새 필드 추가" 필드명 입력란도 동일하게 자동 확장되는가
- [ ] 읽기 모드에서 긴 필드명이 `...`으로 잘리지 않고 줄바꿈되어 표시되는가

### SlotEditModal — 커스텀 필드 / 메모 입력

- [ ] 커스텀 텍스트 필드가 한 줄로 시작하는가 (최소 높이 44px)
- [ ] 텍스트 입력 시 높이가 늘어나는가
- [ ] 엔터(줄바꿈) 입력 시 높이가 늘어나는가
- [ ] 기존 데이터가 있는 필드를 열었을 때 초기 높이가 내용에 맞게 표시되는가
- [ ] 기존 내용이 첫 줄부터 표시되는가 (스크롤 위치 맨 위)
- [ ] 메모(달력에 표시됨) 입력란도 동일하게 동작하는가

## 2. 읽기모드 칩 — 긴 텍스트 줄바꿈

배경: 비고 등 긴 텍스트가 `whitespace-nowrap`으로 인해 칩에서 잘렸음.

- [ ] 슬롯 등록 후 읽기모드 카드에서 긴 텍스트가 줄바꿈되어 표시되는가
- [ ] 줄바꿈(Enter) 입력한 내용이 읽기모드에서도 줄바꿈으로 표시되는가
- [ ] 짧은 텍스트(전화번호, 금액 등)는 여전히 한 줄로 표시되는가

## 3. 메모 placeholder 변경

배경: "메모 (선택)" → "달력에 표시됨"으로 변경하여 해당 내용이 달력에 노출됨을 사용자가 인지하도록 함.

- [ ] 슬롯 등록 모달 메모 입력란의 placeholder가 "달력에 표시됨"으로 표시되는가
- [ ] useDynamicFields 모드(커스텀 필드 모드)에서도 동일하게 표시되는가
- [ ] 회원선택 모드(member select mode)에서도 동일하게 표시되는가

## 4. OptionValueType — 회차 추가

배경: 드롭다운 옵션의 유형에 "회차" 추가 (단위: 회).

- [ ] AdminPage 커스텀 필드 편집 시 선택지 유형에 "회차"가 표시되는가 (금액, 수량, 인원 다음)
- [ ] "회차" 선택 시 단위가 "회"로 표시되는가
- [ ] 회차 값 입력 후 저장하면 읽기모드에서 "N회"로 표시되는가

## 5. 개별회원모드 중복 배정 방지

배경: A회원이 이미 등록된 슬롯에서 A회원으로 또 등록을 시도했을 때 저장 직전에서야 에러가 떴음. 입력 자체를 사전에 차단하도록 개선.

### useDynamicFields (커스텀 필드) 모드

- [ ] 이름 입력란에 이미 등록된 회원명을 입력하는 순간 나머지 입력란이 숨겨지는가
- [ ] "이미 이 슬롯에 배정된 회원입니다." 경고 메시지가 표시되는가
- [ ] 저장 버튼이 비활성화(disabled)되는가
- [ ] 이름을 지우면 경고가 사라지고 입력란이 다시 나타나는가

### 회원선택 모드 (dropdown)

- [ ] 이미 등록된 회원은 선택 드롭다운에 표시되지 않는가
- [ ] 모든 회원이 배정된 경우 "모든 회원이 이미 배정되어 있습니다" 메시지만 보이고 메모 입력란이 숨겨지는가
- [ ] 회원 선택 전에는 메모/추가필드 입력란이 보이지 않는가
- [ ] 유효한 회원을 선택하면 메모/추가필드 입력란이 나타나는가

## 6. 스케줄 등록 모달 스크롤 동작

배경: 모달 위에서 마우스 스크롤 시 배경 페이지가 스크롤됐음. 모달 내부는 스크롤, 배경은 고정되어야 함.

- [ ] 모달이 열려 있는 상태에서 모달 바깥 영역에서 스크롤해도 배경 페이지가 움직이지 않는가
- [ ] 모달 내부 콘텐츠 영역에서 스크롤하면 모달 내용이 스크롤되는가
- [ ] 모달을 닫으면 배경 페이지의 스크롤 위치가 유지되는가
