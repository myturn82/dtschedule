## 1. 날짜 선택 모달 (스크롤 휠 피커)

배경: 월/주/일 뷰 타이틀을 클릭하면 연/월(/일)을 스크롤로 골라 바로 그 날짜로 이동할 수 있는 기능 추가.
이전/다음 버튼도 타이틀 좌우로 재배치하고 월간 뷰의 불필요한 연도 표시는 제거함.

- [ ] 월간 뷰에서 "7월"(제목) 클릭 → 모달이 열리고 연/월 두 컬럼만 표시됨
- [ ] 연도 컬럼에서 위로 스크롤(마우스 휠 위로) → 중앙 값이 더 큰(다음) 연도로 바뀜
- [ ] "확인" 클릭 → 헤더가 선택한 연/월로 즉시 이동
- [ ] "취소" 또는 배경 클릭 → 변경 없이 모달만 닫힘
- [ ] 주간 뷰에서 날짜 범위 텍스트 클릭 → 연/월/일 세 컬럼 표시, 월을 2월로 스크롤하면 일 목록이 28/29일까지만 표시(31일 선택 중이었다면 자동 보정)
- [ ] 일간 뷰에서 "7월 7일" 클릭 → 연/월/일 선택 후 "확인" → 정확히 그 날짜·요일로 이동
- [ ] 모바일 폭에서 모달이 화면 안에 잘 들어오고 잘리지 않음 (배경 애니메이션에 의한 포지셔닝 버그 수정됨)
- [ ] 좌우 이동 버튼 크기가 "7월" 텍스트 크기에 맞게 줄어들어 있음(월/주/일 공통)

크기·톤이 다르게 보이던 문제 정리.

- [ ] 모바일 폭에서 아바타 옆 사용자명이 보이지 않음 (PC(sm 이상)에서는 그대로 표시)
- [ ] 우측 아이콘(공유/알림/피드백/아바타) 간격이 좁아짐
- [ ] 공유 버튼이 모바일에서 알림 벨과 동일한 정사각형 아이콘 버튼으로 표시됨 (PC에서는 기존처럼 "공유" 텍스트 포함 pill 버튼)
- [ ] "전체 검색" 입력창 왼쪽 아이콘이 색 없는 회색 SVG 돋보기로 표시됨(이모지 아님)
- [ ] "스케줄"/"대시보드" 탭 높이가 모바일에서 더 낮게 표시됨 (PC는 기존 높이 유지)

## 3. 미운영 시간대 X 표시 제거

배경: 관리자콘솔 요일별 스케줄 규칙에서 "미운영" 처리한 시간대는 모바일에서 X 대신
빈 칸으로 표시. 날짜별 휴무(override)·휴관은 기존대로 X/휴관 유지.

- [ ] 월간 뷰: 요일별 "미운영" 규칙으로 닫힌 시간대는 모바일에서 X 없이 빈 칸(데스크톱은 "CLOSE" 텍스트 유지)
- [ ] 주간 뷰: 동일하게 요일별 미운영 시간대는 X 없음
- [ ] 특정 날짜를 "휴무" 처리(override)한 시간대는 기존처럼 X 표시 유지 (회귀 없음)
- [ ] 휴관(공휴일/일요일)은 기존처럼 "휴관" 표시 유지 (회귀 없음)
