## 1. 요일 숨김 기능

배경: 조직마다 운영하지 않는 요일을 달력에서 숨길 수 있도록 설정 기능 추가

- [ ] 관리자 콘솔 → 조직 설정 탭 → "요일 숨김" 섹션이 보이는지 확인
- [ ] 일요일(일) 버튼 클릭 시 어두운 색으로 선택 상태 변경 확인
- [ ] 저장 버튼 클릭 후 설정이 유지되는지 확인 (페이지 새로고침)
- [ ] **[DB 확인]** `tenants.settings.hidden_days` 배열에 선택한 요일 번호(0=일) 저장 확인
- [ ] 스케줄 화면 → 월간 뷰에서 선택한 요일 컬럼이 숨겨지는지 확인
- [ ] 스케줄 화면 → 주간 뷰(시간대별)에서 선택한 요일 컬럼이 숨겨지는지 확인
- [ ] 스케줄 화면 → 월간 뷰(일자별)에서 선택한 요일 날짜 셀이 숨겨지는지 확인
- [ ] 스케줄 화면 → 주간 뷰(일자별)에서 선택한 요일 컬럼이 숨겨지는지 확인
- [ ] 숨김 요일이 없을 때 모든 요일이 정상 표시되는지 확인
- [ ] 토요일·일요일 동시 숨김 시 평일만 표시되는지 확인
- [ ] 숨김 해제(버튼 재클릭 → 저장) 후 해당 요일이 다시 표시되는지 확인

## 2. 회원관리 UI 개선

- [ ] 관리자 콘솔 → 회원관리 → 데스크탑에서 이메일·전화번호 옆에 연필 아이콘 표시 확인
- [ ] 모바일에서 이름 아래 이메일·전화번호 옆 연필 아이콘 표시 확인
- [ ] 이메일 아이콘 클릭 시 인라인 편집 폼 열림 확인
- [ ] 모바일에서 테이블 우측이 잘리지 않고 가로 스크롤 작동 확인

---

## 3. 범례 링크 및 브랜드 아이콘

- [ ] 관리자 콘솔 → 범례 관리 → 새 항목 추가 시 "링크 URL" 입력 필드 표시 확인
- [ ] URL 입력 후 저장 시 범례 항목 옆에 외부 링크 아이콘(↗) 표시 확인
- [ ] 스케줄 화면 범례에서 URL이 있는 항목 클릭 시 새 탭으로 링크 이동 확인
- [ ] 아이콘 선택 드롭다운에서 인스타그램(핑크 원) 아이콘 표시 확인
- [ ] 아이콘 선택 드롭다운에서 네이버블로그(초록 N) 아이콘 표시 확인
- [ ] 수정 폼에서도 URL 입력 필드 표시 확인

## 4. 기타 수정

- [ ] 일자별 달력(월간·주간 일자별 뷰) 등록 버튼에서 드롭다운 화살표 제거 확인
