## 5. 다크모드 `<select>` 드롭다운 가시성 수정

배경: 다크모드에서 관리자콘솔 조직 선택 등 `<select>` 드롭다운을 열면 팝업(옵션 목록) 배경이 브라우저 기본값(흰색)으로 렌더링되어 밝은 글자색과 겹쳐 텍스트가 안 보이던 문제. `color-scheme` CSS만으로는 일부 브라우저에서 select 팝업에 반영되지 않아, `.dark select`/`.dark select option`에 배경·글자색을 직접 지정하는 방식으로 수정.

- [ ] 다크모드에서 관리자콘솔 조직 선택 드롭다운을 열었을 때 옵션 목록 배경이 어둡게 표시되고 조직명이 읽힘
- [ ] 다크모드에서 회원관리 탭의 역할/권한 select, 날짜 규칙 탭의 색상/시간 select 등 다른 select들도 옵션 목록이 정상적으로 보임
- [ ] 라이트모드에서는 기존과 동일하게 흰 배경에 어두운 글자로 표시됨 (회귀 없음)
- [ ] 스케줄 페이지 등 관리자콘솔 외 다른 화면의 select도 다크모드에서 정상 표시됨
