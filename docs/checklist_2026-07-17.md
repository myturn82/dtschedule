## 1. 오리지널 포인트 컬러 글씨색 및 토요일 배경색 통일

배경: 오리지널(기본) 프리셋 선택 시 스케줄 셀 글씨가 붉게 보이던 문제와 토요일 셀 배경색이 평일과 달랐던 문제를 수정. AI 예약 모달 메모 필드 리셋 누락도 함께 수정.

- [ ] 오리지널 프리셋 조직에서 스케줄 셀의 글씨색이 붉지 않고 기본 보조 텍스트 색상으로 표시됨
- [ ] 토요일 셀 배경색이 평일과 동일하게 표시됨 (plus 병합 표시 로직은 유지)
- [ ] 다른 포인트 컬러(미드나잇·포레스트 등) 조직에서 토요일·글씨색 변화 없음 (회귀)
- [ ] 자연어 예약 모달(✨)에서 등록 완료 후 메모 필드가 빈 값으로 리셋됨

## 2. 모바일 스케줄 헤더 레이아웃, 햄버거 사이드바 겹침, 일자별 등록 오류 수정

배경: 모바일에서 월/주/일 전환 버튼 위치·역할 칩 잘림·햄버거 사이드바 겹침 등 레이아웃 문제와 일자별 뷰에서 역할 분리 조직 팀원 등록이 조용히 실패하던 버그를 수정.

- [ ] 모바일에서 월/주/일 전환 버튼이 날짜 타이틀 옆(우측)에 작게 표시됨
- [ ] 역할이 여러 개인 조직에서 역할 표시 칩이 잘리지 않고 가로 스크롤로 확인 가능함
- [ ] 시간별/일자별 토글 아이콘이 SVG 모노크롬으로 표시됨 (이모지 아님)
- [ ] 창 폭 1024px 미만에서 햄버거 메뉴 열면 딤 배경(백드롭)이 표시되고, 클릭 시 닫힘
- [ ] 역할 분리 조직에서 일자별 뷰의 "+ 등록" 드롭다운으로 배정 등록 시 정상 등록됨 (**핵심 버그 픽스**)
- [ ] 역할 미분리 조직에서 일자별 뷰 등록도 동일하게 정상 동작함

## 3. 캘린더 UI 아이콘 정리 및 날짜 이동 모달 클릭 선택

배경: 역할 칩과 세그먼트 간 구분선 추가로 시각 분리 강화, 날짜 이동 모달의 스크롤 방식 외에 클릭으로도 선택 가능하도록 개선, 헤더 버튼 데스크톱 라벨 추가.

- [ ] 역할 표시 칩 영역과 시간별/일자별 세그먼트 사이에 구분선이 보임
- [ ] 헤더 날짜 타이틀 클릭 시 날짜 이동 모달이 열림
- [ ] 날짜 이동 모달에서 연/월/일 항목을 **클릭**하면 스크롤 없이 바로 선택됨 (기존 스크롤도 동작)
- [ ] 데스크톱(lg+)에서 헤더의 AI 등록·공유·알림 버튼에 텍스트 라벨이 함께 표시됨
- [ ] 모바일에서는 해당 버튼이 아이콘만 표시됨
- [ ] AI 등록 버튼(✨) 아이콘이 이모지가 아닌 SVG 모노크롬으로 표시됨

## 4. 역할 다중 바표시 지원 및 관리자콘솔 역할 표시방식 3택1 정리

배경: 바표시 역할이 여러 개일 때 각 역할을 구분된 색상 세로 바로 표시하도록 개선. 임의 색상 대신 조직 포인트 컬러 팔레트를 자동 생성해 테마와 항상 어울리도록 함. 관리자콘솔도 위자드와 동일한 3택1 세그먼트로 정리.

- [ ] 바표시(indicator_bar) 역할이 2개 이상인 조직에서 월간 뷰 셀에 역할별 색상으로 구분된 세로 바가 균등 분할로 표시됨
- [ ] 주간 뷰에서도 동일하게 다중 바가 표시됨
- [ ] 일자별(리스트) 뷰에서 역할별 컬러 보더(왼쪽 테두리)가 표시됨
- [ ] 자유입력(비회원) 배정도 role_id 기준으로 매칭되어 바가 정상 표시됨
- [ ] 바표시 역할이 1개인 조직에서 기존과 동일하게 단일 컬러 바가 표시됨
- [ ] 관리자콘솔 → 역할 설정에서 표시방식이 미분리 / 분리 / 바표시 3택1 세그먼트로 변경됨
- [ ] 분리와 바표시가 동시에 켜지는 상태가 더 이상 발생하지 않음 (세그먼트는 1개만 선택 가능)
- [ ] 세그먼트 변경 후 저장 시 DB에 정상 반영됨 **[DB 확인]**

## 5. 모바일 스케줄 헤더 UI 개선 (조직명 표시 + 검색바 토글)

배경: 모바일에서 현재 조직명이 화면에 표시되지 않아 맥락 파악이 어려웠음. AppHeader에 조직명을 표시하고, 돋보기 버튼으로 검색바를 토글 표시하도록 개선.

- [ ] 모바일(sm 미만)에서 앱 헤더에 조직명이 표시됨
- [ ] 헤더의 돋보기(🔍) 버튼을 클릭하면 검색 입력바가 펼쳐짐
- [ ] 다시 돋보기 버튼(또는 닫기)을 클릭하면 검색바가 닫힘
- [ ] ScheduleHeader(달력 영역 상단)에서 조직명이 제거되어 중복 표시가 없음
- [ ] 달력 월 네비게이션(이전/다음 화살표 + 월 표시)이 모바일에서 가운데 정렬됨
- [ ] 데스크톱(sm+)에서 달력 월 네비게이션은 기존 좌측 정렬 유지됨
